# API create using TypeScript & Prisma

## Use NPM Packages

- Express
- TypeScript
- Ts-node
- Prisma
- Cors
- Dotenv
- Cookie-parser
- Express-async-handler
- Express-validator
- jsonwebtoken
- Bcryptjs
- morgan -D
- nodemon -D

## Routes

#### Post

- GET : /posts
- POST : /posts
- GET : /posts/userId=id
- GET : /posts/:id
- PUT : /posts/:id
- DELETE : /posts/:id
- POST : /posts/bulk
- DELETE : /posts/bulk
- GET : /posts/:id/user
- GET : /posts/:id/comments

#### User

- GET : /users
- POST : /users
- GET : /users/:id
- PUT : /users/:id
- DELETE : /users/:id
- POST : /users/bulk
- DELETE : /users/bulk

##### Comments

- GET : /comments
- POST : /comments
- GET : /comments/:id
- PUT : /comments/:id
- DELETE : /comments/:id
- POST : /comments/bulk
